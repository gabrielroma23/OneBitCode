Conhecendo o CSS

Existem três formas principais de aplicar o CSS (Cascading Style Sheets) ao HTML para estilizar páginas web:

1. Inline CSS: Estilo aplicado diretamente dentro dos elementos HTML usando o atributo style.

Este é um texto estilizado com CSS inline.


2. Internal CSS (ou Embutido): dentro do html no topo da página.


External CSS (ou Externo): Estilos definidos em um arquivo CSS separado que é linkado ao documento HTML.
